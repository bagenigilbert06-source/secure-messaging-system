────────────────────────────────────────────────────────────────────────────────

  • currentView: 'auth' | 'student' | 'admin'
  • isAdmin: boolean
  • studentName: string

Student Dashboard State:
  • searchQuery: string
  • selectedCategory: string
  • selectedItem: Item | null
  • showClaimForm: boolean
  • claimProof: string
  • submitting: boolean

  • activeTab: 'dashboard' | 'inventory' | 'messages' | 'settings'

Filtering Logic:
  • useMemo for search + category filtering

  • Real-time as user types
